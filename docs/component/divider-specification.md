## 概要

Dividerコンポーネントは、コンテンツ間にセマンティックな区切り線を提供するUIコンポーネントです。`<hr>` 要素を使用した水平方向の1pxの線で、視覚的にセクションを分離します。

## インポート

```typescript
import { Divider } from '@zenkigen-inc/component-ui';
```

## 基本的な使用方法

```tsx
import { Divider } from '@zenkigen-inc/component-ui';

export function SectionList() {
  return (
    <div className="flex flex-col gap-4">
      <p>セクション 1</p>
      <Divider />
      <p>セクション 2</p>
    </div>
  );
}
```

## Props

このコンポーネントは Props を持ちません。固定のスタイルで描画されるシンプルなコンポーネントです。

## スタイル仕様

| プロパティ | Tailwind クラス | 説明                                   |
| ---------- | --------------- | -------------------------------------- |
| 高さ       | `h-px`          | 1px の線                               |
| 幅         | `w-full`        | 親要素の幅いっぱいに広がる             |
| ボーダー   | `border-0`      | `<hr>` のデフォルトボーダーをリセット  |
| 背景色     | `bg-uiBorder01` | デザインシステムの境界線カラートークン |

## 使用例

### セクション間の区切り

```tsx
<div className="flex flex-col gap-4">
  <p>セクション 1</p>
  <Divider />
  <p>セクション 2</p>
  <Divider />
  <p>セクション 3</p>
</div>
```

### リスト項目間の区切り

```tsx
<div className="flex flex-col" role="list">
  <div role="listitem">項目 1</div>
  <Divider />
  <div role="listitem">項目 2</div>
  <Divider />
  <div role="listitem">項目 3</div>
</div>
```

## アクセシビリティ

- `<hr>` 要素を使用しており、暗黙的に `role="separator"` を持ちます。
- スクリーンリーダーはセクションの区切りとして認識します。
- 追加の ARIA 属性は不要です。

## 技術的な詳細

### 実装について

- `<hr>` 要素を使用したセマンティック HTML。
- `border-0` で `<hr>` のデフォルトスタイルをリセットし、`h-px` + `bg-uiBorder01` で 1px の線を描画。
- Props を持たず、外部からのスタイルカスタマイズは不可。設計方針として、一貫した見た目を保証します。
- `clsx` は使用していません（動的なクラス生成が不要なため）。

## 注意事項

1. 外部からのスタイルカスタマイズはできません。デザインシステムの一貫性を保つための設計方針です。
2. 水平方向のみ対応しています。垂直方向の区切り線が必要な場合は別途対応が必要です。
3. `bg-uiBorder01` カラートークンはテーマに依存します。
