## 概要
リスト形式の複数の選択肢の中から1つを選択できるコンポーネントです。Listと組み合わせて使用します。

## 設計の原則 （Do）

### 選択肢が4つ以上ある場合に使用します
選択肢が4つ以下の場合は、代わりにRadioButtonの使用を検討してください。

## 推奨しないこと（Don't）

### 選択肢が2つの時は基本的に使用しない
選択肢が2つで更にはっきり示せる場合(Yes/Noなど)は1つのCheckBox、または、Toggleにまとめます。
ただし、Selectの中の選択肢の数が状況に応じて変動するケースで場合によっては選択肢が2つになることはあります。また、表などのスペースの制約が強い時に、二者択一の選択をあえてSelectにするという可能性はあります。
