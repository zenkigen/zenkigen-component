## 概要
ボタンを押すというユーザーインタラクションに応じて、リスト形式の複数の選択肢によるパネルが開くコンポーネントです。

## 設計の原則 （Do）

### Listと組み合わせて使用します
選択肢にはListを使用します。
- 基本的にメニューに使われます
- 操作後にアクションが発生します(ページ遷移など)
- (メニューの)その他、絞り込みや並べ替えの操作で使われます

## 推奨しないこと（Don't）

### Selectと混同しない
以下の場合はSelectを使用します。
- 基本的にフォームに使われます
- 基本的に選択肢から選択するときに使用します
- 他のボタンをクリックするまで選択内容は実行・送信されません
