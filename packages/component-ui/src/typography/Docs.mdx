Typography（タイポグラフィ）は、文字のスタイル、サイズ、間隔、階層を定義し、ユーザーにとって読みやすく、一貫性のあるビジュアル体験を提供する ための重要な要素です。デザインシステムにおいて、Typography は単なるフォントの指定にとどまらず、可読性・視認性・アクセシビリティを高め、ブランドのアイデンティティを表現する 役割を担います。

## Heading

Heading のスタイルはユーティリティで適用できるほか、[Heading](/docs/components-heading--docs) コンポーネントでも同様の表現が可能です。

<Canvas of={TypographyStories.Heading} />

#### **使い方**

- className に `typography-h*` を指定して使用します。

```html
<div className="typography-h1">見出しタイトル</div>
```

## Body

<Canvas of={TypographyStories.Body} />

#### **使い方**

- className に `typography-body****` を指定して使用します。

```html
<div className="typography-body12regular">あのイーハトーヴォのすきとおった風、夏でも底に冷たさをもつ青いそら、...</div>
```

## Label

<Canvas of={TypographyStories.Label} />

#### **使い方**

- className に `typography-label****` を指定して使用します。

```html
<div className="typography-label11regular">ユーザー名</div>
```
