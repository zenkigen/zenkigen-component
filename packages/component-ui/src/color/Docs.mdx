# Color Tokens

Design Token（デザイントークン）は、デザインのスタイル（色、タイポグラフィ、スペーシング、シャドウなど）をコードとして定義し、再利用しやすくするための仕組みです。これにより、一貫性のあるUIデザインを提供できます。
デザイントークンは [**Primitive Token（プリミティブトークン）**](#primitive-token) と [**Semantic Token（セマンティックトークン）**](#semantic-token) の 2 つの種類に分かれます。

<div className="bg-uiBackground02">
  こんにちは。この要素の背景が、Semantic/Background uiBackground02 で着色されます。
</div>
<div className="border border-uiBorder01">
  こんにちは。この要素のボーダーが、Semantic/Border uiBorder04 で着色されます。

export function ColorItem({ colorKey, value, description, rawValue }) {

const [copiedName, setCopiedName] = useState(false);
const [copiedValue, setCopiedValue] = useState(false);

const handleCopyName = async () => {
try {
await navigator.clipboard.writeText(colorKey);
setCopiedName(true);
setTimeout(() => setCopiedName(false), 2000);
} catch (err) {
console.error('Failed to copy: ', err);
}
};

const handleCopyValue = async () => {
try {
await navigator.clipboard.writeText(value);
setCopiedValue(true);
setTimeout(() => setCopiedValue(false), 2000);
} catch (err) {
console.error('Failed to copy: ', err);
}
};

const isDeletionCandidate = deletionCandidates.has(colorKey);
const isConsolidationCandidate = consolidationCandidates.has(colorKey);

return (

<div className="flex flex-col items-left gap-2 flex-shrink-0">
  <div
    className="relative flex w-full p-1 justify-end items-end"
    style={{ backgroundColor: value, height: 170 }}
  >
    {rawValue ? (
      <div className="flex bg-white rounded-sm px-1 py-0.5">
        <Icon name="link" size="small" color="icon01" />
        <span style={{ fontSize: '12px', fontWeight: 'bold' }}>{rawValue.replaceAll('$Colors.', '')}</span>
      </div>
    ) : null}
    {isDeletionCandidate ? (
      <div
        className="absolute inset-0 flex items-center justify-center"
        style={{ backgroundColor: 'rgba(255, 255, 255, 0.6)' }}
      >
        <span style={{ color: '#d92b57', fontSize: 120, fontWeight: 'bold', lineHeight: 1 }}>✕</span>
      </div>
    ) : null}
    {isConsolidationCandidate ? (
      <div
        className="absolute inset-0 flex items-center justify-center"
        style={{ backgroundColor: 'rgba(255, 255, 255, 0.4)' }}
      >
        <span style={{ color: '#d97706', fontSize: 100, fontWeight: 'bold', lineHeight: 1 }}>△</span>
      </div>
    ) : null}
  </div>
  <div className="flex flex-col gap-1 text-left pr-1" style={{ minWidth: 170 }}>
    {isDeletionCandidate ? (
      <div style={{ color: '#d92b57', fontWeight: 'bold', fontSize: '12px' }}>✕ 削除候補（未使用）</div>
    ) : null}
    {isConsolidationCandidate ? (
      <div style={{ color: '#d97706', fontWeight: 'bold', fontSize: '12px' }}>△ 統合候補（lib のみ使用）</div>
    ) : null}
    <div className="flex items-center">
      <div className="typography-label11bold">{colorKey}</div>
      <IconButton
        onClick={handleCopyName}
        size="small"
        title="カラー名をコピー"
        icon={copiedName ? 'check' : 'copy'}
        variant="text"
      />
    </div>
    <div className="flex items-center">
      <div className="typography-label11regular text-gray-500">{value}</div>
      <IconButton
        onClick={handleCopyValue}
        size="small"
        title="カラー値をコピー"
        icon={copiedValue ? 'check' : 'copy'}
        variant="text"
      />
    </div>
    {description ? (
      <div className="typography-label14regular" style={{ fontSize: '14px', maxWidth: 150 }}>
        {description}
      </div>
    ) : null}
  </div>
</div>
); }

## Primitive Token

Primitive Token は、デザインの基礎となる原子的な値 を定義します。これは特定の用途を持たない、純粋なスタイル値を表します。

- 生の値（数値やカラーコードなど） を定義する
- 直接 UI に適用せず、Semantic Token によって参照される
- デザイントークンの管理をしやすくする
- グローバルに再利用可能

## Semantic Token

Semantic Token は、コンポーネントや UI の意味に応じたデザインスタイル を定義します。これは、Primitive Token を組み合わせて使います。

- 意味（セマンティクス）に基づいて命名される
- コンポーネントや UI の文脈に応じたスタイルを表現する
- デザイン変更時に一括で更新しやすい
- ブランディングやテーマ適用を柔軟に管理できる
