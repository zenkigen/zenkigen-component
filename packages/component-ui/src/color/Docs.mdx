# Color Tokens

Design Token（デザイントークン）は、デザインのスタイル（色、タイポグラフィ、スペーシング、シャドウなど）をコードとして定義し、再利用しやすくするための仕組みです。これにより、一貫性のあるUIデザインを提供できます。
デザイントークンは Primitive Token（プリミティブトークン） と Semantic Token（セマンティックトークン） の 2 つの種類に分かれます。

### Code Example

- Color Token は Tailwind の色系ユーティリティ（text / bg / border / ring / fill / stroke など）から参照できます。

```html
<div className="text-text02">こんにちは。この要素の文字が、Semantic/Text text02 で着色されます。</div>
<div className="bg-uiBackground02">
  こんにちは。この要素の背景が、Semantic/Background uiBackground02 で着色されます。
</div>
<div className="border border-uiBorder01">
  こんにちは。この要素のボーダーが、Semantic/Border uiBorder04 で着色されます。
</div>
```

<div style={{ width: '100%', maxWidth: 'none' }}>

export function ColorItem({ colorKey, value }) {
  const [copiedName, setCopiedName] = useState(false);
  const [copiedValue, setCopiedValue] = useState(false);

const handleCopyName = async () => {
try {
await navigator.clipboard.writeText(colorKey);
setCopiedName(true);
setTimeout(() => setCopiedName(false), 2000);
} catch (err) {
console.error('Failed to copy: ', err);
}
};

const handleCopyValue = async () => {
try {
await navigator.clipboard.writeText(value);
setCopiedValue(true);
setTimeout(() => setCopiedValue(false), 2000);
} catch (err) {
console.error('Failed to copy: ', err);
}
};

return (

<div className="flex flex-col items-left gap-2 flex-shrink-0">
  <div className="w-full" style={{ backgroundColor: value, height: 170 }} />
  <div className="flex flex-col gap-1 text-xs text-left pr-1" style={{ minWidth: 170 }}>
    <div className="flex items-center">
      <div className="typography-label11bold">{colorKey}</div>
      <IconButton
        onClick={handleCopyName}
        size="small"
        title="カラー名をコピー"
        icon={copiedName ? 'check' : 'copy'}
        variant="text"
      />
    </div>
    <div className="flex items-center">
      <div className="typography-label11regular text-gray-500">{value}</div>
      <IconButton
        onClick={handleCopyValue}
        size="small"
        title="カラー値をコピー"
        icon={copiedValue ? 'check' : 'copy'}
        variant="text"
      />
    </div>
  </div>
</div>
); }

## Primitive Token

Primitive Token は、デザインの基礎となる原子的な値 を定義します。これは特定の用途を持たない、純粋なスタイル値を表します。

### Primitive/Colors

<div style={{ overflowX: 'auto', width: '100%', minHeight: 260 }}>
  <div className="flex gap-1">
    <ColorItem colorKey="black" value={tokens.colors.black} />
    <ColorItem colorKey="white" value={tokens.colors.white} />
  </div>
</div>

### Primitive/Colors/Blue

<div style={{ overflowX: 'auto', width: '100%', minHeight: 260 }}>
  <div className="flex gap-1">
    {Object.entries(tokens.colors.blue).map(([key, value]) => (
      <ColorItem key={key} colorKey={key} value={value} />
