## 概要

検索結果やカテゴリ一覧などリスト化された縦長ページを複数に分割し、次へ・前へを含めた関連ページへのリンクでページを移動するために使用します。

## 設計の原則（Do）

### 大量のアイテムがある場合に使用します

検索結果やカテゴリ一覧など、1ページに記載するには大量すぎるアイテムを複数ページに分けて表記する場合に、ページ間の移動を可能にします。

### ページネーションの範囲を指定できます

現在のページの左右に何桁表示するか、開始と終了のページ番号に隣接して表示するかを指定することができます。

### スペースに応じて、Selectを使ってページを選択できるようにします

ページ数が多くなると、全部を表示するのは無理があります。スペースに応じて、Selectを使ってページをコンパクトに表示するようにします。
![設計の原則 その3](/storybook/images/pagenationSelect_01.jpg)

## 推奨しないこと（Don’t）

### アイテム(内容)が少ない場合は使用しない

内容が少ない場合はPaginationを使用せず、1ページにまとめておきます。
