## Code Examples

```tsx
import { useState, useCallback } from 'react';
import { EvaluationStar } from '@zenkigen-inc/component-ui';

const EvaluationExample = () => {
  const [rating, setRating] = useState(3);

  const handleChange = useCallback((newRating: number) => {
    setRating(newRating);
  }, []);

  return (
    <div className="flex flex-col gap-2">
      <p className="typography-label16regular text-text02">体験の評価</p>
      <EvaluationStar value={rating} isEditable onChangeRating={handleChange} />
    </div>
  );
};
```

## 概要

水平に配置された一連のグラフィックシンボル(星)を使用して、評価を表示します。 ユーザーはボタンを通じて評価を入力することができます。

## 設計の原則 （Do）

### 評価は5段階、または3段階で表示します

値は、5つの場合、0から5までの感情尺度(0が最低、5が最高)、3つの場合、0から3までの感情尺度(0が最低、3が最高)で、選択された評価の数です。

## 推奨しないこと（Don’t）

### 同一の評価軸の場合は星の数を項目ごとに変更しない

スターレーティングは、5つ、または3つの星を使用します。 同一の評価軸の場合に、星の数を項目ごとに増やしたり減らしたりしてはいけません。 同一の評価軸の場合は、他の項目と同じ星の数で評価します。
