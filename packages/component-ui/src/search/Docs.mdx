## 概要

ユーザーがアプリ全体で情報をすばやく見つけられるようにするナビゲーション方法です。ユーザーが検索バーまたは検索ビューのテキストフィールドにクエリを入力すると、関連する結果が表示されます。

## 設計の原則（Do）

### 検索結果がない場合は、ユーザーが情報にたどり着くための解決策を提示します

エンプティステートメントは、表示する情報がない場合に表示します。ただ、情報がないことを示すだけでなく、利用者が欲しい情報にたどり着くための解決策を提示してください。
![設計の原則 その1](/storybook/images/search_01.jpg)

## 推奨しないこと（Don’t）

### データ量が少ない、または限られている場合は使用しない

情報が単純で、1つのビューで簡単に見つけることができる場合は使用しないでください。
