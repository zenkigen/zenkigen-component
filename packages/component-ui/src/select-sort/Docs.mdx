## 概要

テーブルのヘッダーをクリックすることで、指定されたキーに基づいて行を昇順・降順に並べ替えるUIコンポーネントです。

## 設計の原則（Do）

### Listと組み合わせて使用します

選択肢にはListを使用してください。Listには「昇順で並べ替え」「降順で並べ替え」を使用します。

### 操作後にアクションが発生します

選択肢の中から1つを選択すると、それに伴い並べ替えのアクションがすぐに実行されます。

## 推奨しないこと（Don’t）

### 複数条件で絞り込みをする場合は使用しない

条件の掛け合わせで実行をさせたい場合は[Select](/docs/components-select--docs)を使用する。
